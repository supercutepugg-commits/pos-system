# POS System

Next.js 기반 POS 관리 시스템.

## 개발 시작하기

로컬 개발 환경 세팅(패키지 설치, 개발용 Supabase 프로젝트 생성, 환경 변수)은 [docs/dev-environment.md](./docs/dev-environment.md) 참고.

```bash
npm install
cp .env.example .env  # 값 채우기
npm run dev
```

http://localhost:3000 에서 확인합니다.
